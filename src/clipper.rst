编译安装
**********************

按照如下操作在FENGSim中编译clipper2

* 首先克隆FENGSim。 ::
  
    git clone https://github.com/OpenDigitalTwin-Dev/FENGSim.git

* 在 ``FENGSim/toolkit/Geometry/clipper2/`` 中有一个install脚本，运行该脚本安装clipper2。 ::
  
    cd FENGSim/toolkit/Geometry/clipper2/
    ./install

clipper2要清楚以下内容，第一个是代码的获取，第二个是库依赖关系的配置和安装，第三个是编译和安装。
